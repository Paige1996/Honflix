<h2> :pushpin: 주요 기능

<h3>* [로그인 페이지] 로그인, 회원가입, 로그인 유효성 검사,

<h3>* [메인 페이지] 유튜브 카테고리 별로 크롤링, 자취생 필수 웹사이트 추천 링크
