<h2> :pushpin: 주요 기능

<h3>* 로그인 페이지 : 로그인, 회원가입, 로그인 유효성 검사

<h3>* 메인 페이지 : 유튜브 카테고리 별로 크롤링

<h3>* 메인 디테일 페이지 : 댓글 등록, 댓글 삭제, 댓글 좋아요, 북마크, 각 영상별로 추천 영상 10가지
